VSCode终端字体设置：Nerd Fonts

发表于 2021-02-27 更新于 2024-03-07
作者 Yushu
1 分钟阅读

在vscode上设置适配vim和oh-my-zsh的字体，从而解决字体不能正确显示的问题

方法如下：

1.安装Powerline fonts

这是一个字体包的安装，也可以直接百度去下单个字体。

Debian/Ubuntu等基于Linux的环境：

1

sudo apt-get install fonts-powerline

其他环境，如mac：

1
2
3
4
5
6
7
8

# clone 下载
git clone https://github.com/powerline/fonts.git --depth=1
# install 安装
cd fonts
./install.sh
# clean-up a bit 删除安装包
cd ..
rm -rf fonts

（可选）查看字体库

1

cd ~/Library/Fonts

可以看到有很多带for powerline的字体，感兴趣也可以自己尝试一下，在vscode的使用格式如下文。

2.设置

在设置栏中输入：

DroidSansMono Nerd Font

参考：

VScode终端字体设置 - 简书 (jianshu.com)
